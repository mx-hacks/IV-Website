Correr el server con python3 -m http.serve en la carpeta de proyecto

Adentro de app están los archivos html de casa sección

Adentro de Assets están las imágenes y los css de cada sección (Todos los CSS convergen en un main.css)

Algunos elementos están insertados mediante controladores (Los que se repiten mucho)
se insertan con {{  }} y se debe poner el nombre del controlador y la propiedad, como si fuera un objeto {{ Coche.ruedas }}
Los componentes están en la misma carpeta que los html y básicamente son una función con las propiedades.
